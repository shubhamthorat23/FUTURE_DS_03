## 📌 Project Overview
This project focuses on analyzing user behavior across a marketing funnel:
**Visitors → Leads → Customers**

The objective is to understand conversion performance and identify where users drop off in the funnel.

---

## 🎯 Objectives
- Analyze user journey from visit to purchase  
- Measure conversion performance  
- Track user activity over time  
- Identify top-performing products, brands, and categories  

---

## 📊 Dashboard Features
- Funnel Overview (Visitors → Leads → Customers)  
- User Activity Over Time  
- Revenue Analysis (Purchase-based)  
- Top Products  
- Top Brands  
- Top Categories  

---

## 🛠 Tools Used
- Tableau  
- Data Cleaning  
- Data Visualization
